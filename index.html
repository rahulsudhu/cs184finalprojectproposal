Summary

Our final project will be an interactive grid based 2D colorful smoke simulation governed by the Navier-Stokes equations. The Navier Stokes equations describe the conservation of physical properties (like mass, momentum, and energy) to model fluid flows.To solve 2D incompressible steady-state Navier-Stokes equations, approximations and assumptions are made so that the appropriate solutions can be achieved. We will write shaders that compute this directly on the GPU. We will compute velocity, density, and pressure fields. The user will be able to directly generate smoke with user defined radius, vorticity, and dissipation rates.

Problem Description 

Our goal is to render 2D smoke simulations based on the Navier Stokes equations on an Eulerian grid. We will create shaders which compute grid values directly on the GPU. On mouse clicks and drags, we should see bright colorful smoke generated. Solid boundaries will be used to confine the fluid with the option of setting periodic boundary conditions. We will compute velocity, pressure, density, and divergence using WebGL shaders. The user should be able to control radius, vorticity with a curl paramet, velocity dissipation and density dissipation,


Goals and Deliverables

We plan on delivering an interactive colorful smoke simulation webpage using WebGL and three.js. We will also deliver videos walking through the different capabilities of the simulation.

We hope to add different creative effects like bloom, diffuse shading, and random color generation for the smoke. We also hope to avoid aliasing caused by the grid using a blur. If we have time we will try to add solid 2D objects that the fluid interacts with.

We will measure quality by the speed of the simulation and accuracy of the solver in accordance to real-life fluid dynamics behavior and the aesthetic quality of the simulation.
